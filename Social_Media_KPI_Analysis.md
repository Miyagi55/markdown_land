## 3. Instagram — Growth Metrics & KPIs

### Virality Analysis System

| Metric | Threshold |
|---|---|
| Minimum viral views | **20x average views** |
| Minimum engagement rate for tracking | **5%+** |
| Engagement formula | (likes + comments + shares + bookmarks) / views |
| Bookmarks | "Most underestimated type of engagement" |
| Insane viral benchmark | If avg = 10K views, viral = 200K; **insane = 2,000,000 views** (recreate immediately) |
| Reel age for sheet inclusion | Don't add reels under **72 hours** old |
| Metric consistency | Don't mix 5-day-old metrics with 24-hour metrics |

### Pattern Analysis Tracked Per Reel

| Element | Options |

| Lighting | Natural / Artificial |

| Sound | Viral trend / ASMR / Music / Voiceover / Sexual Sounds |
| Hook Strength | High / Medium / Low |
| Sexuality Point | Tracked |
| Sexuality Level | Tracked |
| Caption | Tracked |
| Location | Tracked |
| Outfit Reveal/Transition | Tracked |
| Filter Used | Tracked |
| Potential for Reuse | Tracked |

**Per-model requirement:** **6 separate virality sheets** if you have 6 models

### Performance Tracking System

| Code | Meaning |
|---|---|
| **Green** | Worked on **3+ accounts** → add to winning vault |
| **Red** | Failed on **5+ accounts** → add to graveyard, never reuse |

### Shadowban & Ban Incidents

| Incident | Period | Detail |
|---|---|---|
| IG Ban Wave #1 | June 2025 | Massive bans, days with only **4 new paid subs** |
| IG Ban Wave #2 | September 2025 | Main IG account shadowbanned (lowered sub influx in 2nd half) |
| Widespread log-out issue | October 2025 | All users logged out, IG + Threads bug affecting entire platform |
| **Status September evaluation** | September 2025 | "No bans or shadowbans reported" (before the Oct shadowban) |

### Shadowban Recovery Protocol

| Step | Action |
|---|---|
| 1 | Account logged out from all devices |
| 2 | SMS code → "logging in wasn't you" message |
| 3 | If still locked out → turn off VPN/proxy, try different device |
| 4 | **Maximum 3 attempts** from any devices |
| 5 | After 3 failures → wait **72 hours** with zero attempts |
| 6 | If still failed → wait another **72 hours** |
| 7 | Use existing IG account on phone → long press → "Add new account" |

**Prevention Measures:**
- **Meta Verified profiles:** Much lower chance of bans/shadowbans
- **Threads-linked accounts:** All logged-out accounts had NO Threads profiles
- **Never** run attempts from PC or public Wi-Fi (airport, coffee, gym)
- **Never** have two agency accounts following/interacting with each other
